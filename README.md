# Webpack & React - Skeleton

This is just the basic skeleton that I use to create a new react apps.  I've included all of the things that I like:

* React
* Webpack
* Sass
* React-Router
* Babel support
* Pushstate-server

and more

## Install

    $ npm install

## Build

    $ webpack

## Run

    $ npm start
